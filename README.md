# Clear Search Buffer

Clear the search buffer when pressing RETURN

### Install

Easy installing with [Vundle](https://github.com/gmarik/Vundle.vim)

Simply add to you plugins in .vimrc

    Bundle 'johnantoni/vim-clear-search-buffer'

Launch VIM and run:
  
    :BundleInstall
